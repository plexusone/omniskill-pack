# Release Notes - v0.2.0

**Release Date:** 2026-07-05

## Overview

This release renames the package from `omniagent-skills` to `omniskill-pack` for consistency with the omniskill ecosystem naming convention.

## Installation

```bash
go get github.com/plexusone/omniskill-pack@v0.2.0
```

Requires Go 1.26+ and omniskill v0.9.0+.

## Highlights

- **Renamed to omniskill-pack** - Module path changed from `github.com/plexusone/omniagent-skills` to `github.com/plexusone/omniskill-pack`

## Breaking Changes

### Module Rename

Update your imports:

```go
// Before
import skills "github.com/plexusone/omniagent-skills"

// After
import skills "github.com/plexusone/omniskill-pack"
```

The `Pack.Name()` method now returns `"omniskill-pack"` instead of `"omniagent-skills"`.

## What's Changed

### Refactored

- Module path renamed from `github.com/plexusone/omniagent-skills` to `github.com/plexusone/omniskill-pack`
- `Pack.Name()` returns `"omniskill-pack"`

### Documentation

- Updated README badges and shields for new repository name
- Updated LICENSE file

### Dependencies

- Bump `github.com/plexusone/omniskill` from v0.7.0 to v0.9.0
- Bump `github.com/grokify/oscompat` from v0.3.0 to v0.4.0

## Migration Guide

1. Update your `go.mod`:
   ```bash
   go get github.com/plexusone/omniskill-pack@v0.2.0
   ```

2. Update imports in your Go files:
   ```bash
   # Using sed
   find . -name "*.go" -exec sed -i '' 's|plexusone/omniagent-skills|plexusone/omniskill-pack|g' {} +
   ```

3. Run `go mod tidy` to clean up old dependencies

## Included Skills

- **Utility (10):** notion, slack, trello, gh-issues, summarize, openai-whisper-api, xurl, healthcheck, blogwatcher, gemini
